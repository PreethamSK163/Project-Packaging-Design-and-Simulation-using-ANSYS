<h2>🔍 Overview</h2>

- Launched **ANSYS Electronics Desktop (AEDT)** and set up a complete **flip-chip BGA package** model — including silicon die, solder bumps, substrate, and encapsulant.
- Assigned material properties and configured thermal power sources within the die to simulate real operating conditions — preparing the model for thermal analysis.

<h2>📝 Stage Details</h2>

**Task 1 — Getting Started with ANSYS AEDT** &nbsp;|&nbsp; `AEDT` `GUI` `Project Setup`

Explored the ANSYS Electronics Desktop environment — a comprehensive multiphysics simulation platform for electromagnetic, thermal, and circuit analysis. Familiarized with the ribbon-based interface, project modules (HFSS, Icepak, Maxwell, Q3D), and navigation through geometry creation, meshing, simulation setup, and results tabs.

**Task 2 — Flip-Chip BGA Package Setup** &nbsp;|&nbsp; `Flip-Chip BGA` `3D Modeling` `Layer Stack`

Constructed a complete flip-chip BGA package model — defining substrate, die, solder bumps, and encapsulant layers with precise dimensions and thickness. Assigned materials to each component and organized the model hierarchy. Configured the die-to-substrate flip-chip connection with solder bumps underneath, preparing the structure for thermal and electrical simulation.

**Task 3 — Material Definitions & Thermal Power Sources** &nbsp;|&nbsp; `Material Properties` `Thermal Conductivity` `Power Sources`

Assigned material properties to every package layer — thermal conductivity, specific heat, and density for silicon die, underfill resin, substrate, and metallic vias. Configured thermal power generation within the die to simulate heat produced under operating conditions. Set up temperature monitors at critical points for comprehensive data collection during simulation.
